Binary Search Tree in Java with insert, delete, search and in-order, pre-order, post-order traversals — Level 3 Task 3 of Java Development Internship at Codveda Technology

Here's how to run it — this one needs multiple terminal windows:
# Compile all files
javac Server.java ClientHandler.java Client.java

//Terminal 1 — start the server first
  
  -java Server
  

//Terminal 2 — first client
  
  -java Client
  

//Terminal 3 — second client (open another terminal)
  
  -java Client
  

Each client will be asked for a username, then they can chat. Messages from one client are broadcast to all others in real time. Type exit to leave.
